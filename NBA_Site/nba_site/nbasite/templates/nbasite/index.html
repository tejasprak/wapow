Evaluation of Existing Statistics

A comprehensive evaluation of current advanced NBA statistics that attempt to isolate individual player ‘value’/’contribution to winning’. This introduces a method to evaluate the correlation between these advanced statistics and the most important value for any player - win percentage.

Regularizing Adjusted Plus-Minus

A description and implementation of the process used to calculate Adjusted Plus-Minus, which is widely considered to be one of the best evaluation metrics. APM by itself is not predictive by any means, so we use ridge regression (a Bayesian technique) to improve the predictiveness and accuracy.

Finding Undervalued Players

Using historical rolling Regularized Adjusted Plus-Minus to build and train a model to predict player value, and then applying it to find undervalued and overvalued NBA players from the 2019-20 NBA season. Specifically, creating a list of players that are most conducive to winning.
